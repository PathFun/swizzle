# 表单方法（form）

FormRender v1.x 使用了状态内置的模型，所以外部对表单的所有操作都依赖于 form 实例提供的方法。 本篇会 cover 以下这些常用 api：

form.submit
form.setValues
form.setValueByPath
form.setSchemaByPath

以及以下的生命周期

beforeFinish
onFinish

我们写一个最常用的场景：加载一个已经填写完成的表单，从服务端异步获取数据（这里使用 mock）；修改表单并提交新数据给服务端
